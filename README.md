# 🦷 DentEst Dental Clinic – Website

This project is a **service and pricing webpage for DentEst Dental Clinic**, built using HTML and CSS. It presents available dental services, pricing information, and allows users to conveniently book appointments online.

---

## 📌 Features

- ✨ Service pricing display
- 🦷 Detailed descriptions of dental treatments
- 💶 Pricing information for each service
- 📅 Online booking integration (Setmore)
- 🧭 Simple navigation (Home, Services, Contact)
- 📱 Responsive design (mobile-friendly)

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (Setmore iframe integration)

---

This project uses **Setmore** for appointment scheduling.  
Each booking button redirects users to the corresponding service page.

## 🎯 Purpose

The goal of this project is to:
- provide a clean and user-friendly interface  
- showcase dental services clearly
